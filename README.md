CFour
=====

A super simple connect four type game engine and a CLI frontend for using it.

Written in Python (3.6+)

## Tests

In a python 3.6+ virtual environment:
```
pip install -r requirements.txt
```

Then execute the tests:
```
py.test
```

## CLI Frontend

```
python cli_four.py
```
