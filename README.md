CFour
=====

A super simple connect four type game engine and a CLI frontend for using it.

Written in Python (3.7+)

## Tests

This project uses pipenv:

```
pdm install
```

Then execute the tests:
```
pdm run py.test
```

## CLI Frontend

```
pdm run python cli_four.py
```
